# 练习

你现在需要存储一个学生的信息。需要存储它的姓名（用字符代替），性别（int :0 男 1 女），学号（int）。显然通过上面的学习，你需要使用一个结构体存储它！

1. 设计该结构体，并且构造一个变量来存储这个学生的信息。
2. 针对该结构体，设计一个打印函数，将该学生的信息打印出来，预期的输出如下：

```bash
name: X, gender: 0, id: 123456
```
