# Eazy-C-Master-Tutorial

吉林大学无线电爱好者协会2024C语言速通课程，为单片机培训服务

注意，本节课程并非严格意义上的C语言课程，为了能够让学生使用Arduino + ESP-IDF进行开发，我们会在课程中穿插一些C++的内容。
